What Is SPK

Spk. Is a voice controlled News application using Alan AI. Alan's API can request data from News API. Then with the information from News API the Alan API can give you what ever you ask for regarding the news and the user controls everything in the app using their voice. The app is extremely easy to integrate and the goal of the app is to give the user a easy experience of receiving information.
